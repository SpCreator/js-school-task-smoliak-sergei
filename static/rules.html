Правила игры

Цель игры

Кликая левой клавишей мыши по цветным кубикам Вы набираете очки. Наберите за отведенное время минимальное кол-во очков (300)

Запуск/Пауза/Возобновление/Перезапуск игры

Для запуска игры нажмите кнопку "Start"

Для паузы игры, во время игры нажмите "Pause"

Для снятия игры с паузы нажмите "Resume"

Для перезапуска игры нажмите "New Game"

Очки

Кубики разного цвета приносят разное количество очков:

Красный -1

Синий +1

Черный +2

Желтый +3

Зеленый +4

Уровни и время

Максимальный уровень в игре 12

Переход на новый уровень происходит при условии набора минимального количества очков и лишь по истечении общего времени (60 секунд)

При переходе на новый уровень общее время игры на уровне уменьшается на 5 секунд

По истечении времени, если минимальное количество очков не было набрано - игра заканчивается


Таблица результатов

После завершения игры Вы можете добавить свои достижения в таблицу результатов. Для этого по завершении игры впишите свое имя в поле "Name" и кликните по кнопке "Save"

Таблица результатов отображает результаты игроков в алфавитном порядке основываясь на имени игрока.

Результаты игры можно очистить кликнув по кнопке "Clean up"
